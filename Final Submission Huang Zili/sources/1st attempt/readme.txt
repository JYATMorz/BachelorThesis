Unzip the compressed file.
Install NodeJS if it's not installed before or its version is under 12.
Open the command prompt (Press ‘win + R’, type in 'cmd' in pop-up 'Run' application and click 'OK'.)
Check for local IPv4 address through 'ipconfig'
Navigate to '\1st Attempt\LocalGame\server'.
Enter 'node server.js'
Select 'Allow' with default selection if Windows Firewall shows up.
After 'Listening on 8081' shows up, open the web browser and type in the address to access the client.
	(Address: local IPv4 address : 8081, for example: 192.198.1.30:8081)
